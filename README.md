## What JokoBox Does

JokoBox runs in the browser and helps prepare SVG files for laser cutting with a workflow built around immediate visual feedback. It is meant for makers, operators, and design teams that need quick iteration, readable previews, and files that can move cleanly into cutting software.

Instead of splitting layout work across multiple tools, it brings live rendering, nesting, unit conversion, and view controls into one responsive interface. That makes it easier to shape laser-ready geometry, inspect it, and export it without extra handoffs.

## How to Use It

You can use JokoBox to create or refine laser-cut layouts directly in the browser:

1. Open the application.
2. Add or edit your shapes and part geometry.
3. Watch the preview update as you work.
4. Switch views if you need a different inspection mode.
5. Apply nesting to organize parts for output.
6. Export SVG files for use in CorelDraw or laser machine workflows.

For best results, review labels, path cues, and units before exporting the final file.
